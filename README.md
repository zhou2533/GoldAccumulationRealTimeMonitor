# 🏦 多银行积存金实时监测脚本（Tampermonkey）
一款油猴 Tampermonkey 专用脚本，实时监控9大银行积存金、伦敦金、京东实时金价，支持持仓盈亏统计、自定义价格预警、桌面通知、悬浮轮播球、深浅色主题、窗口拖拽。

## ✨ 核心功能
### 📈 行情监控
- 覆盖 9 家主流银行积存金：
  工商银行 / 农业银行 / 中国银行 / 建设银行 / 招商银行 / 兴业银行 / 民生银行 / 浙商银行 / 广发银行
- 额外标的：京东24小时黄金、伦敦金（美元报价）
- 自动轮播跑马灯展示全部实时报价，涨跌颜色区分

### 💰 持仓管理 & 盈亏自动计算
- 手动录入每家银行持仓克重、成本单价
- 自动计算：
  当前市值 | 总成本 | 单日盈亏 | 总盈亏 | 日收益率 | 总收益率
- 汇总全部持仓整体盈亏数据，一目了然

### 🔔 价格预警 + 系统桌面通知
- 自定义**上涨突破价 / 下跌跌破价**提醒
- 浏览器桌面弹窗通知，附带标的名称、当前价、目标价、涨跌幅度
- 5分钟默认防重复冷却机制，避免消息轰炸
- 支持启用/停用单条提醒、一键删除提醒规则

### 🎛 个性化设置
- 自定义行情自动刷新间隔（10s ~ 300s）
- 悬浮球轮播切换间隔自定义
- 深色 / 浅色双主题一键切换
- 面板、悬浮球自由拖拽，位置自动记忆
- 快捷键 `Alt + G` 快速显示/隐藏主面板

### 🎯 交互细节
- 悬浮球模式：收起面板后自动轮播展示关键金价
- 接口失败自动缓存上次价格，标记异常数量提醒
- 手动一键刷新按钮 + 加载旋转动画
- 操作轻提示 Toast
- 油猴菜单快捷入口：切换面板、手动刷新、打开设置

## 📦 安装方式
### 前置要求
1. 浏览器安装 Tampermonkey（油猴）扩展
   - Chrome / Edge：应用商店搜索「Tampermonkey」安装
   - Firefox：附加组件商店安装 Tampermonkey
2. 确保浏览器开启**桌面通知权限**（用于价格提醒弹窗）

### 安装步骤
1. 新建脚本：油猴面板 → 添加新脚本
2. 清空默认模板全部代码
3. 复制本项目 `gold-monitor.user.js` 完整源码粘贴进去
4. `Ctrl + S` 保存，脚本自动启用
5. 任意网页页面刷新即可出现金价面板

## ⚙ 使用教程
### 1. 录入持仓
1. 点击面板底部 ⚙ 设置按钮
2. 下滑到「我的持仓」
3. 选择银行，填写持仓克数、成本单价，点击添加
4. 支持修改、删除已有持仓
5. 点击底部「保存持仓与设置」生效，自动计算盈亏

### 2. 添加价格提醒
1. 设置面板下滑至「价格提醒」
2. 选择标的、选择≥上涨 / ≤下跌、输入触发价格
3. 点击添加生成提醒规则
4. 可单独开关提醒、删除单条规则
5. 价格触碰阈值自动弹出系统通知

### 3. 基础配置项说明
| 配置项 | 说明 |
|--------|------|
| 刷新间隔（秒） | 全局行情自动拉取周期，范围 10~300 |
| 悬浮球间隔（秒） | 收起面板后轮播切换标的速度，范围 3~60 |
| 主题切换 | 深色模式（默认）/ 浅色模式 |

### 4. 快捷键 & 油猴菜单
- 快捷键：`Alt + G` 快速显示/隐藏主面板
- 油猴右键菜单：
  - 切换显示面板
  - 手动刷新数据
  - 打开设置面板

## 🧩 脚本权限说明
脚本头部 `==UserScript==` 配置权限用途：
```js
// @grant        GM_xmlhttpRequest     // 跨域请求金价接口
// @grant        GM_addStyle            // 注入页面样式
// @grant        GM_registerMenuCommand // 油猴右键菜单
// @grant        GM_getValue / setValue // 本地持久化配置（持仓、提醒、位置、主题）
// @grant        GM_notification        // 桌面价格弹窗通知
// @connect      jin.20021002.xyz      // 金价数据源接口域名
