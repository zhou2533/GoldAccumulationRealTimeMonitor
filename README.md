# 黄金积存金实时监测-价格提醒版
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一款基于油猴（Tampermonkey）的浏览器脚本，实时监测9家银行积存金及国际金价，支持持仓管理、盈亏自动计算、自定义价格提醒（桌面通知），帮助黄金投资者及时掌握价格动态。

## 功能特性
### 📊 核心监测
- 实时监测9家银行积存金价格：民生银行、浙商银行、兴业银行、工商银行、招商银行、广发银行、农业银行、建设银行、中国银行
- 同步监测京东24h金、伦敦金（国际金价）
- 自动计算涨跌幅、当日涨跌额，价格变动实时可视化

### 📦 持仓管理
- 支持添加多家银行的黄金持仓（克数+成本价）
- 自动计算持仓市值、日盈亏、总盈亏、收益率等核心指标
- 持仓数据本地持久化存储，刷新不丢失

### 🔔 价格提醒
- 自定义价格提醒规则（上涨突破/下跌跌破目标价）
- 桌面通知推送，支持5分钟冷却防骚扰
- 多币种适配（人民币/美元）

### ✨ 交互体验
- 面板支持拖拽移动、自定义位置保存
- 深色/浅色主题切换
- 悬浮球快捷查看价格，支持自动轮播
- 自定义刷新间隔（10-300秒），失败自动重试
- 接口失败时沿用缓存数据并标记提醒

## 安装使用
### 前置条件
- 浏览器安装油猴插件（Tampermonkey）：[Chrome](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) / [Edge](https://microsoft.com/en-us/microsoft-edge/add-ons/Microsoft-Edge-Home/Microsoft-Edge-Details/Tampermonkey) / [Firefox](https://addons.mozilla.org/zh-CN/firefox/addon/tampermonkey/)

### 安装步骤
1. 复制本脚本完整代码
2. 打开油猴插件，点击「新建脚本」
3. 清空默认代码，粘贴复制的脚本内容
4. 保存并启用脚本（快捷键 `Ctrl+S`）
5. 打开任意百度页面（脚本匹配域名：`https://www.baidu.com/*`），即可看到监测面板

### 基础操作
1. **面板控制**：点击面板右上角「×」隐藏面板，悬浮球点击可重新展开
2. **刷新设置**：面板底部可手动刷新、调整刷新间隔
3. **持仓添加**：点击「⚙ 设置」→ 「我的持仓」添加/修改/删除持仓数据
4. **价格提醒**：点击「⚙ 设置」→ 「价格提醒」添加提醒规则（上涨≥/下跌≤目标价）
5. **主题切换**：面板标题栏点击「☀/☾」切换深浅主题

## 数据说明
- 数据来源：京东金融、招商银行公开接口，其他银行数据来自聚合接口
- 接口限流：2次/秒/IP，失败自动重试1次
- 缓存机制：接口请求失败时，沿用上次成功获取的数据并标记⚠
- 提醒冷却：同一提醒规则5分钟内仅推送一次通知

## 常见问题
1. **脚本不显示？**
   - 确认脚本已启用，且当前页面为百度域名（`https://www.baidu.com/*`）
   - 检查油猴插件是否允许脚本运行，刷新页面重试

2. **价格不更新？**
   - 检查网络连接，确认接口未被屏蔽
   - 查看面板底部状态提示，失败次数>0时会显示⚠，可手动点击刷新

3. **桌面通知不弹出？**
   - 浏览器设置中允许百度网站发送通知
   - 确认提醒规则已启用，且价格达到触发条件
